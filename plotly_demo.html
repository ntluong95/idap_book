1. Two categorical columns (Stacked bar plot):
import plotly.express as px
import pandas as pd
from palmerpenguins import load_penguins

penguins = load_penguins().dropna()

fig = px.bar(penguins, x="species", color="island", barmode="relative")
fig.update_layout(title="Penguins by Species and Island")
fig.show()

This will create a stacked bar plot showing the relative frequencies of islands for each species.

  1. Three or more columns (Scatter plot with facets):
fig = px.scatter(
    penguins,
    x="bill_length_mm",
    y="flipper_length_mm",
    color="species",
    symbol="species",
    facet_col="island",
    title="Bill Length vs Flipper Length by Species and Island",
)
fig.show()

This creates a scatter plot of bill length vs flipper length, with different colors and symbols for each species, and separate facets for each island.

  1. Time series:

First, we need to aggregate the data:

penguins_count = penguins.groupby(["year", "island"]).size().reset_index(name="count")

fig = px.line(
    penguins_count,
    x="year",
    y="count",
    color="island",
    line_dash="island",
    title="Number of Penguins by Year and Island",
)
fig.show()

This creates a line plot showing the number of penguins over time for each island, with different colors and line styles for each island.

These Plotly Express functions create interactive plots that you can zoom, pan, and hover over for more information. They’re also easily customizable with additional parameters if you need to adjust aspects like the layout, axis labels, or color scheme.
